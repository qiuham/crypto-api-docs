# Retrieving Your Trades

To retrieve trades on OKX, make a request to the `GET /api/v5/sprd/trades` endpoint.

---

# 检索您的交易

要检索您在 OKX 上的交易，您应该向 `GET /api/v5/sprd/trades` 发出请求。
