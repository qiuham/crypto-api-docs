# Submitting an Order

To submit an order to a Spread's Order Book, make a request to the `POST /api/v5/sprd/order` endpoint.

---

# 提交订单

要向 某个Spread 的订单簿提交订单，您应该请求 `POST /api/v5/sprd/order` 。
