# Margin Account Cancel OCO (TRADE)

## API Description[​](/docs/margin_trading/trade/Margin-Account-Cancel-OCO#api-description "Direct link to API Description")

Cancel an entire Order List for a margin account.

## HTTP Request[​](/docs/margin_trading/trade/Margin-Account-Cancel-OCO#http-request "Direct link to HTTP Request")

DELETE /sapi/v1/margin/orderList

## Request Weight[​](/docs/margin_trading/trade/Margin-Account-Cancel-OCO#request-weight "Direct link to Request Weight")

**1(UID)**

## Request Parameters[​](/docs/margin_trading/trade/Margin-Account-Cancel-OCO#request-parameters "Direct link to Request Parameters")

Name| Type| Mandatory| Description  
---|---|---|---  
symbol| STRING| YES|   
isIsolated| STRING| NO| for isolated margin or not, "TRUE", "FALSE"，default "FALSE"  
orderListId| LONG| NO| Either `orderListId` or `listClientOrderId` must be provided  
listClientOrderId| STRING| NO| Either `orderListId` or `listClientOrderId` must be provided  
newClientOrderId| STRING| NO| Used to uniquely identify this cancel. Automatically generated by default  
recvWindow| LONG| NO| The value cannot be greater than `60000`  
timestamp| LONG| YES|   
  
Additional notes:

  * Canceling an individual leg will cancel the entire OCO

# 取消杠杆账户 OCO 订单(TRADE)

## 接口描述[​](/docs/zh-CN/margin_trading/trade/Margin-Account-Cancel-OCO#接口描述 "接口描述的直接链接")

取消杠杆账户订单列表。

## HTTP请求[​](/docs/zh-CN/margin_trading/trade/Margin-Account-Cancel-OCO#http请求 "HTTP请求的直接链接")

DELETE /sapi/v1/margin/orderList

## 请求权重[​](/docs/zh-CN/margin_trading/trade/Margin-Account-Cancel-OCO#请求权重 "请求权重的直接链接")

**1(UID)**

## 请求参数[​](/docs/zh-CN/margin_trading/trade/Margin-Account-Cancel-OCO#请求参数 "请求参数的直接链接")

名称| 类型| 是否必需| 描述  
---|---|---|---  
symbol| STRING| YES|   
isIsolated| STRING| NO| 是否逐仓杠杆，"TRUE", "FALSE", 默认 "FALSE"  
orderListId| LONG| NO| `orderListId` 或 `listClientOrderId` 必须被提供  
listClientOrderId| STRING| NO| `orderListId` 或 `listClientOrderId` 必须被提供  
newClientOrderId| STRING| NO| 用户自定义的本次撤销操作的ID(注意不是被撤销的订单的自定义ID)。如无指定会自动赋值。  
recvWindow| LONG| NO| 不能大于 `60000`  
timestamp| LONG| YES|   
  
其他注意点:

  * 取消单个 leg 将取消整个 OCO 订单。
