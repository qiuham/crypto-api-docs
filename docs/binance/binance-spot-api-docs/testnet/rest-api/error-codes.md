# Error Codes

* Any endpoint can return an ERROR



Sample Payload below:
    
    
    {  
        "code": -1121,  
        "msg": "Invalid symbol."  
    }  
    

  * Specific error codes and messages are defined in [Errors Codes](/docs/binance-spot-api-docs/testnet/errors).
