# Check Server Time

## API Description[​](/docs/derivatives/options-trading/market-data#api-description "Direct link to API Description")

Test connectivity to the Rest API and get the current server time.

## HTTP Request[​](/docs/derivatives/options-trading/market-data#http-request "Direct link to HTTP Request")

GET `/eapi/v1/time`

## Request Weight[​](/docs/derivatives/options-trading/market-data#request-weight "Direct link to Request Weight")

**1**

## Request Parameters[​](/docs/derivatives/options-trading/market-data#request-parameters "Direct link to Request Parameters")

NONE

## Response Example[​](/docs/derivatives/options-trading/market-data#response-example "Direct link to Response Example")
    
    
    {  
      "serverTime": 1499827319559  
    }
